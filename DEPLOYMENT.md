## 🎯 STEP-BY-STEP DEPLOYMENT

### Step 1: Test Locally (2 minutes)
1. Open File Explorer
2. Navigate to `d:\AK\`
3. Double-click `index.html`
4. Your portfolio opens in browser! ✨

### Step 2: Create GitHub Repository (3 minutes)
1. Go to https://github.com
2. Click "+" → "New repository"
3. Repository name: `genai-portfolio`
4. Description: "My Gen AI Portfolio Website"
5. Keep it Public
6. **DON'T** check "Add README"
7. Click "Create repository"

### Step 3: Upload Files (2 minutes)

**Option A: Drag & Drop (Easiest)**
1. On the new repository page, click "uploading an existing file"
2. Drag these files from `d:\AK\`:
   - index.html
   - styles.css
   - script.js
   - README.md
3. Scroll down, click "Commit changes"

**Option B: Using Git (If you have Git installed)**
```bash
cd d:\AK
git init
git add .
git commit -m "Initial commit: Gen AI Portfolio"
git remote add origin https://github.com/YOUR-USERNAME/genai-portfolio.git
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages (1 minute)
1. In your repository, click "Settings" (top right)
2. Scroll down to "Pages" (left sidebar)
3. Under "Source":
   - Branch: Select "main"
   - Folder: Select "/ (root)"
4. Click "Save"
5. Wait 2-3 minutes

### Step 5: Access Your Live Site! 🎉
Your portfolio is now live at:
```
https://YOUR-USERNAME.github.io/genai-portfolio/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

## 🔧 TROUBLESHOOTING

### Issue: Site not loading after 5 minutes
**Solution**: 
1. Go to Settings → Pages
2. Check if "Your site is live at..." message appears
3. Try accessing in incognito/private mode
4. Clear browser cache

### Issue: Changes not showing
**Solution**:
1. Wait 2-3 minutes after pushing changes
2. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
3. Clear browser cache

### Issue: 404 Error
**Solution**:
1. Ensure `index.html` is in root directory (not in a folder)
2. Check repository is Public
3. Verify GitHub Pages is enabled in Settings
